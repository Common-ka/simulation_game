# GameManager

Главный тик-контроллер игры. Периодически (каждую секунду по-умолчанию) вызывает тик обновления ресурсов (`GameTick`), увеличивая валюту и оповещая всех подписчиков.

## Классы и структуры

### `UnclaimedAssets.Core.GameSnapshot`
Структура, содержащая слепок игровых данных.
- **`SoftCurrency`** (`double`): Текущее количество мягкой валюты.

### `UnclaimedAssets.Core.GameManager`
Основной монобех, вешается на единственный GameObject в сцене `Game`.

#### События
- **`public static event Action<GameSnapshot> OnGameStateChanged`**: Статическое событие, к которому могут подписаться UI элементы для получения актуальных данных без прямого знания о `GameManager`.

## Архитектурные Правила
- Управляет состоянием и экономикой, соблюдая строгую изоляцию от UI. 
- Все вычисления в `GameTick` проходят без выделения новой памяти и `GetComponent`.
- Интервал тика настраивается в инспекторе через поле `_tickInterval`.
