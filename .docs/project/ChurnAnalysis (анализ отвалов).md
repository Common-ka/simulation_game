## Прогнозируемые точки отвала (Churn Points)

### 🔴 CP-1: Первый «стоп» — переход к «Складским остаткам» (кат. 4)
**Проблема:** После скоростного открытия кат. 2 и 3 (2–7 минут) игрок впервые встречает ожидание ~33 минуты. Это психологический удар «первой стены» — большинство казуальных игроков закрывают игру именно здесь.
- **Сим-тайминг:** `00:07 → 00:40` (33 минуты без нового контента)
- **Риск отвала:** ⭐⭐⭐⭐⭐ (D0/D1, самый высокий)

### 🔴 CP-2: Длинная пауза перед «Таможенным конфискатом» (кат. 5)
**Проблема:** Почти 2 часа в одной категории. Первый рубеж, где отключённый игрок (AFK) может вернуться и увидеть прогресс — или не вернуться вовсе.
- **Сим-тайминг:** `00:40 → 02:26` (1 ч 46 мин)
- **Риск отвала:** ⭐⭐⭐⭐ (D1, высокий)

### 🟠 CP-3: Ночной переход в «Антиквариат» (кат. 6)
**Проблема:** Открытие происходит примерно в ~7-8 часов реального времени. Большинство игроков зашли утром и к вечеру только добрались до 6-й категории. Если нет пуш-уведомления «Ваша посылка готова!» — игрок уснёт и не вернётся.
- **Сим-тайминг:** `02:26 → 07:44` (5 ч 18 мин)
- **Риск отвала:** ⭐⭐⭐⭐ (D1, конверсия в DAU)

### 🔴 CP-4: «Ювелирный лом» — начало Стены Престижа (кат. 7)
**Проблема:** Переход занял более суток. Игрок первый раз видит «я не открою следующее за сессию». Именно здесь нужно объяснить механику Престижа. Без этого — прямой отвал.
- **Сим-тайминг:** `Day 1, 07:44 → Day 2, 04:11` (20+ часов)
- **Риск отвала:** ⭐⭐⭐⭐⭐ (D2, критичный)

### 🟠 CP-5: «Крипто-фермы» (кат. 8) — гринд в 3 дня
**Проблема:** Игрок понимает, что следующая категория займёт 3 реальных дня. Если Альбом стикеров не даёт мотивации прямо сейчас — смерть сессии для всех, кроме хардкора.
- **Сим-тайминг:** `Day 2 → Day 4` (2 дня 7 часов)
- **Риск отвала:** ⭐⭐⭐ (D3–D5)

### 🟡 CP-6: Порог Престижа — «Знаю что делать, но не хочу сбрасывать»
**Проблема:** Психологический барьер перед первым Престижем. Игрок накопил прогресс и боится потерять. Если игра не показывает ясно «ты получишь ×N», он откладывает — и не возвращается.
- **Сим-тайминг:** Конец Эпохи 1 (Day 14)
- **Риск отвала:** ⭐⭐⭐ (D7–D14)

## Рекомендуемые метрики (Analytics Events)

### Прогрессия категорий
```
event: category_unlocked
params:
  category_id: int (1–10)
  category_name: string
  time_since_start_seconds: int
  epoch: int (1 или 2)
  session_number: int
```
> Позволит построить воронку по каждой категории. Дропы видны как delta между N и N+1.

### Первая встреча со Стеной (кат. 7)
```
event: prestige_wall_reached
params:
  category_id: int
  epoch: int
  time_in_game_seconds: int
  ips_at_moment: float
  shelf_filled: bool
```

### Туториал Черного рынка
```
event: black_market_tutorial_completed
params:
  artifact_id: int       // 1005
  epoch: int
  time_since_start_seconds: int
```

### Случайный дроп ключа
```
event: black_market_key_dropped
params:
  source: string         // "random_box"
  category_name: string
  time_since_start_seconds: int
  epoch: int
```

### Первый завершённый сет стикеров
```
event: sticker_set_completed
params:
  category_name: string
  is_first_ever: bool
  stickers_crafted_count: int
  epoch: int
  time_since_start_seconds: int
```

### Переход к Престижу
```
event: prestige_initiated
params:
  epoch_from: int
  category_reached: int
  lifetime_earnings: float
  pp_earned: float
  prestige_multiplier: float
  stardust_converted: float
  stickers_total: int
  sets_completed: int
  artifacts_owned: int
```

### Смерть сессии
```
event: session_end
params:
  session_duration_seconds: int
  category_at_exit: int
  soft_currency_at_exit: float
  epoch: int
```
> Паттерн `category_at_exit=4 + session_duration < 600` → отвал CP-1.
