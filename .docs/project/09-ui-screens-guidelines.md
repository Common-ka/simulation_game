# Требования к UI (Справочник элементов)
## Unclaimed Assets

Этот документ — шпаргалка для сборки UI в UI Builder. Здесь перечислены все сущности, кнопки и окна, которые **обязательно** должны присутствовать в интерфейсе, чтобы все механики из GDD и монетизации работали корректно.

---

## Архитектура UI Templates (Модульность)
Интерфейс не собирается в едином огромном "монолитном" файле `MainScreen.uxml`.
Используется принцип изоляции (Composition): 
- Каждая логическая зона (`HUD`, `Геймплей`, `Рынок`, `Навигация`) верстается в независимом файле `*Panel.uxml` с прямым импортом глобальных стилей (`Variables.uss`, `Common.uss`, `MainScreen.uss`).
- `MainScreen.uxml` служит исключительно точкой сборки Root-экрана через теги `<ui:Template>` и `<ui:Instance>`.

**Взаимодействие с UIManager:** 
Скрываемые панели (окна) всегда должны содержать CSS класс `.panel` на корневом Visual Element внутри своего UXML. (Например, `HUD` его не содержит, так как не скрывается менеджером, а `Market` или `Upgrades` - содержат).

## 1. Постоянный HUD (Виден всегда) ✅

### 1.1 Шапка (Верхняя панель)
- [ ] Обычная валюта (Soft Currency) — Текстовое значение + [ ] Доход в секунду (IPS) — Текстовое значение (рядом с Soft)
- [ ] Токены (Hard Currency) — Текстовое значение + Кнопка `[+]` (ведет в Магазин)
- [ ] Звёздная пыль (Stardust) — Текстовое значение
- [ ] Золотые ключи (Black Market Keys) — Текстовое значение

### 1.2 Плавающие элементы (Всегда поверх или сбоку)
- [ ] Иконка "Starter Pack" (Отображается, если не куплен)
- [ ] Иконка "Limited Offer" (Дневной спецвыпуск)
- [ ] Иконка/Кнопка активации +30 мин оффлайна (Выбор: реклама / Токены)

### 1.3 Подвал (Навигация)
- [ ] Кнопка вкладки `[Инвентарь]`
- [ ] Кнопка вкладки `[Апгрейды]`
- [ ] Кнопка вкладки `[Крутки / Гача]`
- [ ] Кнопка вкладки `[Стикеры]`
- [ ] Кнопка вкладки `[Магазин]`
- [ ] Кнопка входа на **Чёрный рынок** (с бейджиком-уведомлением, если есть бесплатный прокрут или скидка)

---

## 2. Экран Геймплея (Центральная зона)

- [ ] Инфо-блок текущей категории (Иконка, Название: например, "Складские остатки")
- [ ] Прогресс-бар до следующей категории
- [ ] Кнопка перехода к следующей категории (или с надписью "Слишком дорого")
- [ ] **Витрина (5 слотов):** каждый слот должен уметь показывать иконку предмета, его цветовую ауру (редкость) и показатель дохода.

---

## 3. Основные окна (Вкладки навигации)

### 3.1 Окно Апгрейдов
- [ ] Список доступных улучшений текущей категории
- [ ] В каждой карточке апгрейда: Иконка, Название, Эффект (+20% к доходу), текущая Цена, Кнопка `[Купить]`

### 3.2 Окно поиска предметов (Гача / Коллекции)
- [ ] Анимация/зона Рулетки (перебор предметов)
- [ ] Кнопка `[Крутить рулетку]`
- [ ] Блок крафта стикеров (Показывает цену в пыли и кнопку скрафтить)
- [ ] Зона альбома для просмотра собранных коллекций
- [ ] **Рентген лота (X-Ray):** Кнопка `[Смотреть что внутри (Реклама / 15 Токенов)]`
- [ ] Кнопка `[Обновить лот беслпатно]`

### 3.3 Магазин
Переключение между 5 вкладками:
- [ ] Вкладка `💎 Валюта` (Пакеты Токенов)
- [ ] Вкладка `💰 Ресурсы` (Покупка софт-валюты и пыли за Токены)
- [ ] Вкладка `🔑 Ключи` (Пакеты ключей)
- [ ] Вкладка `⭐ VIP` (Серебро и Золото)
- [ ] Вкладка `🎁 Офферы`

## 4. Окно Чёрного Рынка

- [ ] Цена входа (Отображение стоимости в ключах)
- [ ] Кнопка входа за ключи
- [ ] Кнопка `[Доп. слот артефакта (Реклама / ЯМ)]` (Доступна до открытия контейнера)
- [ ] **Внутри контейнера:** 3 слота предложенных артефактов (Иконка, Описание эффекта)
- [ ] Кнопка `[Забрать предмет]` (под каждым артефактом)
- [ ] Кнопка `[Перекрутить весь лот (Реклама / Токены)]` (Reroll)

## 5. Всплывающие окна (Popups)

### 5.1 Оффлайн доход (Появляется при старте)
- [ ] Текст "Пока вас не было, заработано..."
- [ ] Сумма (Soft Currency)
- [ ] Кнопка `[Забрать]`
- [ ] Кнопка `[Забрать Х2 (Реклама / 10 Токенов)]`

### 5.2 Обновление контракта (Престиж)
*Появляется при попытке открыть последнюю (10-ю) категорию*
- [ ] Заголовок "Государственный конфискат недоступен"
- [ ] Информация о текущем уровне Lifetime Earnings
- [ ] Какой множитель дохода игрок получит после сброса (Permanent Prestige Multiplier)
- [ ] Красный текст-предупреждение: "Все деньги, полки и апгрейды сбросятся!"
- [ ] Главная кнопка `[Подписать (Сброс)]`

### 5.3 Провал крафта
*Когда игрок пытается скрафтить стикер, но не хватает пыли*
- [ ] Сообщение "Не хватает звёздной пыли"
- [ ] Кнопка `[Получить 50 пыли (Реклама / 8 Токенов)]`
- [ ] Кнопка `[Закрыть]`
