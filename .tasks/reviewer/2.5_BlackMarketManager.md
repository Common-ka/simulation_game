# 2.5 — BlackMarketManager: Логика ключей и бонуса IPS

## Название
BlackMarketManager — Система дропа ключей и расчёта BM-бонуса IPS

## Контекст
Система Чёрного рынка (ЧР) является ключевым мета-слоем игры. На данном этапе реализуется **только backend-логика**:
- Дроп ключей при открытии коробок (подписка на `GachaController.OnLootBoxOpened`)
- Хранение количества ключей в `RetainedData`
- Расчёт суммарного IPS-бонуса от артефактов ЧР с наложением хардкапа

UI Чёрного рынка реализуется в отдельной задаче.

## Архитектурные требования

1. **Паттерн:** Singleton MonoBehaviour (`BlackMarketManager.Instance`).
2. **Расположение файла:** `Assets/Scripts/BlackMarket/BlackMarketManager.cs`.
3. **Независимость:** Никаких прямых вызовов UI. Никаких вызовов других менеджеров кроме `SaveManager` и `GachaController` (только подписка на событие).
4. **Кулдаун:** Хранится в `RetainedData.LastKeyDropTimestamp` (Unix seconds, `long`). При проверке использовать `DateTimeOffset.UtcNow.ToUnixTimeSeconds()`.
5. **Кулдаун-константа:** Вынести `7200L` в `private const long KEY_DROP_COOLDOWN_SECONDS`.
6. **Шанс дропа:** Вынести `0.05f` в `private const float KEY_DROP_CHANCE_PERCENT`.
7. **Без `Update()`:** Логика работает только через обработчик события, не через тик.
8. **Newtonsoft.Json:** Для десериализации артефактов (никакого `JsonUtility`).
9. **Загрузка артефактов:** Через `Resources.Load<TextAsset>` из `"GameData/BlackMarketArtifacts"`, выполнить в `Awake()` или через зависимость от `GameDataLoader`. Выбрать простейший вариант — прямой `Resources.Load`.

### [НОВЫЙ] `Assets/Scripts/BlackMarket/BlackMarketManager.cs`
Singleton с логикой:

**Поля:**
```csharp
private const long KEY_DROP_COOLDOWN_SECONDS = 7200L;
private const float KEY_DROP_CHANCE_PERCENT = 0.05f;

private List<BlackMarketArtifact> _artifactsDB = new();
private RetainedData Data => SaveManager.Instance.Data;
```

**`Awake()`:**
- Singleton-инициализация
- Загрузка `BlackMarketArtifacts.json` через `Resources.Load<TextAsset>`
- Десериализация в `_artifactsDB` через `JsonConvert.DeserializeObject<List<BlackMarketArtifact>>`

**`OnEnable()` / `OnDisable()`:**
- Подписка/отписка от `GachaController.OnLootBoxOpened`

**`private void HandleLootBoxOpened(int categoryIndex)`:**
```
1. Получить текущее unix-время: long now = DateTimeOffset.UtcNow.ToUnixTimeSeconds()
2. Проверить кулдаун: if (now - Data.LastKeyDropTimestamp < KEY_DROP_COOLDOWN_SECONDS) → return
3. Проверить шанс: if (Random.value * 100f >= KEY_DROP_CHANCE_PERCENT) → return
4. Обновить кулдаун: Data.LastKeyDropTimestamp = now
5. Вызвать AddKey()
6. Debug.Log("[BlackMarketManager] Найден ключ ЧР! Дроп из коробки.")
```

**`public void AddKey()`:**
```csharp
Data.BlackMarketKeys++;
SaveManager.Instance.Save();
```

**`public double GetBM_IPSBonus()`:**
```
1. Получить список OwnedArtifactIDs из Data
2. Для каждого ID найти артефакт в _artifactsDB
3. Суммировать EffectValue для артефактов с EffectType == "+Total_IPS%"
4. Нормализовать: rawBonus = sum / 100.0
5. Вычислить хардкап: hardCap = (BaseHardCapPercent / 100.0) * (1.0 + PrestigeScaling * Data.PrestigeCount)
   Источник данных → напрямую из _artifactsDB (или хардкод: BaseHardCapPercent = 300, PrestigeScaling = 0.1)
   ВАЖНО: Данные хардкапа должны читаться из GameDataLoader, если он уже инжектирован,
   иначе — значения константами (300 и 0.1) согласно BlackMarketConfig.json.
6. return Math.Min(rawBonus, hardCap)
```

> **Примечание по хардкапу:** Фактическая формула из `03-technical-gdd.md` (раздел 4.5):  
> `maxBmBonus = BaseHardCapPercent / 100.0 * (1 + 0.1 * PrestigeCount)`  
> При 0 Престижей → хардкап = 3.0 (т.е. +300%)
