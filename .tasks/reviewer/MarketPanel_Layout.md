# Название: Верстка интерфейса покупки лотов по макету gameplay.pen

## Контекст
Необходимо перевести дизайн интерфейса покупки лотов и мультипликаторов (LotSelectionMenu) из макета Pencil (`gameplay.pen`) в верстку Unity UI Toolkit (`MarketPanel.uxml` и `Roulette.uss`). Это ключевой элемент монетизации и Гачи, который должен быть перенесен строго по макету (1-к-1), сохраняя киберпанк-стилистику, эффекты свечения и полупрозрачные фоны.

## Связанная документация (ОПТИМИЗАЦИЯ ТОКЕНОВ)
**Внимание Воркеру!** Ты обязан использовать скилл `pencil-to-uitoolkit` и следовать его чеклисту:
1. `.agents/skills/pencil-to-uitoolkit/SKILL.md` (Главная инструкция по снятию метрик из `.pen` и обходу ограничений USS).
2. Читать другие базовые файлы без необходимости запрещено.

## Архитектурные требования
- **Чтение макета Pencil:** Обязательно используй инструменты MCP сервера `pencil` (такие как `mcp_pencil_batch_get` и `mcp_pencil_get_screenshot`) для чтения файла `gameplay.pen`. Самостоятельно найди фреймы, относящиеся к покупке лотов (LotSelectionMenu, OpenLotAction, PriceIndicator), и сними точные метрики (width, gap, padding, background-color, border-radius).
- **Изоляция:** Файл `MarketPanel.uxml` должен быть самостоятельным модулем и использовать только глобальные стили (`Variables.uss`, `Common.uss`) и локальный стиль `Roulette.uss`. Не использовать inline-стили.
- **Ограничения UI Toolkit:** Строго соблюдай ограничения USS, описанные в скилле. Не используй свойство `gap` (заменяй на `margin`), не используй `box-shadow` напрямую, если он недоступен (применяй доступные в проекте методы для теней и свечения `$glow_cyan`). Максимальное внимание на `shadowType outer` и полупрозрачные фоны.
- **Никакого C#:** В рамках таски создается только UXML и USS. Скрипты не трогать.

## Критерии приемки (Acceptance Criteria)
- [ ] Прочитан скилл `pencil-to-uitoolkit/SKILL.md`.
- [ ] Проанализирован файл `gameplay.pen` через MCP инструменты `pencil`, найдены нужные фреймы.
- [ ] Извлечены точные метрики, цвета и отступы из элементов макета.
- [ ] Написан план Flexbox-иерархии перед написанием UXML.
- [ ] Сверстан блок меню лотов (включая центральную кнопку `OpenLotAction` и индикатор `PriceIndicator`), строго следуя отступам из макета.
- [ ] Воспроизведены эффекты свечения (`glow_cyan` / тени / `shadowType outer`) и полупрозрачные фоны с учетом ограничений Unity.
- [ ] Создан или обновлён файл `MarketPanel.uxml` и привязан `Roulette.uss`.
- [ ] В консоли Unity нет ошибок компиляции USS (проверено через `read_console`).
- [ ] Проведена визуальная проверка с помощью `render_ui` и результат совпадает со скриншотом из `gameplay.pen`.
