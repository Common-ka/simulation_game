# Task M3.3.5: Точечная сборка FooterPanel (Нижняя навигация)

**Контекст:**  
Требуется сверстать нижнюю навигационную панель (`FooterPanel.uxml`) для переключения основных вкладок игры. Панель должна быть гибкой (flex) и содержать 6 равнозначных кнопок (Upgrades, Inventory, Gacha, Sticker Album, Shop, Black Market), которые равномерно распределяются по всей доступной ширине. Дизайн переносится из макета `gameplay.pen` (секция `BottomBar`).

**Связанная документация (ОПТИМИЗАЦИЯ ТОКЕНОВ):**
- `.docs/project/09-ui-screens-guidelines.md` (Правила верстки UI и структура)
- `.docs/project/02-asset-conventions.md` (Работа с ресурсами, шрифтами и иконками)
> **Внимание:** Чтение других документов не требуется! Не трать токены на изучение GDD.

**Архитектурные требования:**
1. **Только Верстка (UI Skeleton):** Задача состоит только в написании `.uxml` и `.uss`. C#-скрипты на данном этапе не создаются (это отдельная задача).
2. **Стили и Переменные:** 
   - Все цвета и шрифты строго брать из `Variables.uss` и `Common.uss` (использовать классы или переменные вроде `$glow_cyan`, `$text_main`, `$text_dim`).
   - Кнопки по умолчанию должны иметь фон `#111111E6` (или соответствующую переменную из системы).
   - Для кнопки Black Market используется уникальный стиль обводки и текста (`$glow_purple`).
3. **Верстка Layout (Flexbox):**.
   - 6 кнопок должны равномерно заполнять ширину: обязательно использовать `flex-grow: 1` на каждую кнопку.
   - Внутри каждой кнопки (Иконка + Текстовая группа) элементы центрируются (`justify-content: center`, `align-items: center`), `gap: 24px`, горизонтальный `flex-direction: row`, внутренние отступы `padding: 24px 48px`, обводка (stroke) 2px, скругление (`border-radius`) `20px`.
   - Текстовая группа внутри кнопки (Название + Сабтайтл) выстраивается вертикально (`flex-direction: column`, `align-items: center`), `gap: 4px`.
4. **Типографика:**
   - Названия (Title): Шрифт `Russo One` (переменная `--font-russo-one`), размер 30px, `letterSpacing` 1. Цвет `$text_main`.
   - Подзаголовки (Sub): Шрифт `JetBrains Mono` (переменная `--font-jetbrains-mono`), размер 18px. Цвет `$text_dim`.
   - Иконки: Использовать шрифтовые иконки `Material Icons` (32px, цвет `$glow_cyan`), кроме иконки Black Market (там используется картинка или пустой `VisualElement` с фоном нужной иконки размером 44x44px).

**Критерии приемки (Acceptance Criteria):**
- [ ] Создан/модифицирован `FooterPanel.uxml`. В корневом элементе должны быть подключены стили `Footer.uss` (и общие стили, если требуется).
- [ ] Сверстано ровно 6 кнопок: Upgrades, Inventory, Gacha, Sticker Album, Shop, Black Market.
- [ ] Все кнопки тянутся равномерно (`flex-grow: 1`) и панель не ломается при скейлинге (изменении ширины экрана в UI Builder).
- [ ] Использованы шрифтовые иконки из Material Icons для первых 5 кнопок. У Black Market есть небольшой бейдж (Notification Badge, размер 32x32px, сдвинут через абсолютное позиционирование или отрицательные отступы в верхний правый угол).
- [ ] Для всех текстов, теней и обводок используются глобальные USS переменные проекта. Хардкода цветов в `.uss` следует избегать (только для полупрозрачных фонов, если переменных для них еще нет).
- [ ] Вызов `render_ui` на панель показывает визуальное соответствие макету (кнопки ровные, тексты по центру, цвета правильные).
