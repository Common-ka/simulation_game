# ТЗ 2.5: StickerManager — Удаление AutoCraft

- **Название:** Удаление системы AutoCraft из StickerManager
- **Файл к изменению:** `Assets/Scripts/Gacha/StickerManager.cs`
- **Статус:** 📥 В очереди

---

## Контекст

В игре **не будет** системы AutoCraft. Стикеры крафтит только сам игрок вручную.
Текущая реализация `StickerManager.cs` содержит лишний метод `AutoCraft()` и вспомогательный массив `_rarityOrder`, который используется только в нём.
Их нужно удалить, оставив `CraftCostTable` — он понадобится для ручного крафта в будущей задаче.

## Что удалить

### 1. Метод `AutoCraft()`

```csharp
// УДАЛИТЬ ЭТОТ БЛОК ЦЕЛИКОМ:
public void AutoCraft()
{
    foreach (ItemRarity rarity in _rarityOrder)
    {
        double cost = CraftCostTable[rarity];
        if (cost <= Data.CurrencyStardust)
        {
            int newItemId = UnityEngine.Random.Range(1, 99999);
            HandleNewSticker(newItemId, rarity, "AutoCraft");
            Data.CurrencyStardust -= cost;
            Debug.Log($"[StickerManager] AutoCraft: Crafted sticker (Rarity:{rarity}). Dust spent: {cost}. Remaining: {Data.CurrencyStardust}");
            break;
        }
    }
}
```

### 2. Массив `_rarityOrder`

```csharp
// УДАЛИТЬ (использовался только в AutoCraft):
private static readonly ItemRarity[] _rarityOrder = new ItemRarity[]
{
    ItemRarity.Common, ItemRarity.Uncommon, ItemRarity.Rare,
    ItemRarity.Epic, ItemRarity.Legendary, ItemRarity.Unique
};
```
