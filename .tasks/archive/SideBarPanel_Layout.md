# M3.3.4 — Точечная сборка SideBarPanel (Офферы)

**Контекст:**
Необходимо сверстать правую боковую панель `SideBarPanel.uxml` для отображения быстрых предложений (офферов). Панель содержит "плавающие" элементы, такие как "Starter Pack", "Limited Offer" и бонус "+30 min Offline", которые описаны в UI guidelines. Элементы должны точно соответствовать дизайн-макету из `gameplay.pen`.

**Связанная документация (ОПТИМИЗАЦИЯ ТОКЕНОВ):**
- `.docs/project/09-ui-screens-guidelines.md` (Раздел 1.2 Плавающие элементы)
- `.docs/project/02-asset-conventions.md` (Правила именования)

**Архитектурные требования:**
- Стили должны быть вынесены в отдельный файл `SideBar.uss`.
- Панель использует строгое абсолютное позиционирование для всего блока по правой стороне корневого экрана.
- Внутренняя структура панели — строгий `flex-direction: vertical`.
- Кнопки офферов должны быть независимыми визуальными блоками, чтобы C#-логика могла скрывать/показывать их по отдельности.
- Запрещено использовать хардкод в C# скриптах для стилей — всё оформление задаётся только через UXML и USS.
- В UI Toolkit нет прямой поддержки `box-shadow` для эффекта свечения, поэтому свечение реализуется за счет стилизации обводки (`border-color` и `border-width`), либо использования заранее заготовленных 9-slice текстур, если они предусмотрены проектом. Пока ограничиться стильной яркой обводкой.

**Критерии приемки (Acceptance Criteria):**

- [ ] Создан файл `SideBarPanel.uxml`.
- [ ] Создан файл стилей `SideBar.uss`, который импортируется в `SideBarPanel.uxml`.
- [ ] Корневой элемент `SideBarPanel` имеет классы, задающие `position: absolute; right: 0; top: 150px;` (или привязку по центру/сверху), `width: 220px;`, `flex-direction: vertical;`, `gap: 16px;`, `padding: 16px;`, `align-items: center;`.
- [ ] Сверстаны 3 элемента (оффера) внутри панели. Размеры каждого: `width: 150px; height: 150px;`.
- [ ] **Визуальный стиль карточки оффера:**
  - Фон (`background-color`): `#151515`.
  - Скругление углов (`border-radius`): `12px`.
  - Внутренние отступы: `padding: 12px; gap: 12px;`.
  - Выравнивание внутри карточки: `align-items: center; justify-content: center;`.
  - Обводка (`border-width: 3px`): для Starter Pack и Limited Offer цвет обводки `#FE53BB` (розовый), для Offline Bonus — `#FCEE0A` (желтый).
- [ ] **Контент карточки оффера:**
  - Контейнер под иконку размером `101x105 px`.
  - Текст-название под иконкой: шрифт JetBrains Mono (normal), `font-size: 21px`, выравнивание по центру, цвет текста совпадает с цветом обводки.
- [ ] **Бейдж с таймером (TimerBadge):**
  - Абсолютное позиционирование поверх карточки: `position: absolute; top: 10px; left: 10px;`.
  - Фон бейджа: `#FF0073` (для розовых офферов) или `#FFD500` (для желтого оффера).
  - Скругление углов (`border-radius`): `6px`.
  - Отступы: `padding: 4px 8px;`, `gap: 4px;`.
  - Выравнивание: `flex-direction: row; align-items: center;`.
  - Внутри бейджа: место под иконку часов (`14x14 px`) и текст формата `23:59:59` (`font-size: 12px`, шрифт JetBrains Mono (black), цвет `#FFFFFF` или `#000000` в зависимости от фона).
- [ ] Макет не содержит ошибок и готов к подключению в качестве `<ui:Instance>` на главный экран.
