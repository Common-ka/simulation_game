# Task M3.3.3: ShelfPanel Layout

**Контекст:**
Необходимо сверстать интерфейс витрины (ShelfPanel) для игровых предметов. Нужно создать правильную сетку на 5 слотов поверх фонового изображения полки, настроить метрики и тени. В рамках этой задачи пишется только UI-разметка (UXML + USS), без C#-логики.

**Связанная документация:**
- `.docs/project/09-ui-screens-guidelines.md` (Раздел 2: Экран Геймплея -> Витрина)
- `.docs/project/02-asset-conventions.md`

**Архитектурные требования:**
1. **Изоляция и структура:** Верстка должна располагаться в `ShelfPanel.uxml`. Стили необходимо вынести в отдельный файл `Shelf.uss` и импортировать глобальные переменные (например, `Variables.uss`).
2. **Фон:** В качестве фона полки использовать изображение, лежащее по пути `Assets/Resources/Items/shelf.png`.
3. **Метрики бокса:** Контейнер витрины должен иметь размеры строго **586x539** пикселей.
4. **Отрисовка объема:** Для бокса необходимо настроить тени (свечение), чтобы передать объем. Рекомендуется использовать переменные свечения из проекта (например, `$glow_cyan` или эквивалентные из `Variables.uss`, цвета из Pencil макета: `#63d4d5`).
5. **Сетка слотов (Flexbox):** Внутри контейнера необходимо разместить ровно 5 элементов-слотов (класс `.shelf-slot`). Распределить их по сетке с помощью Flexbox (использовать `justify-content` / `align-items` / `flex-wrap` по ситуации, чтобы они эстетично ложились поверх фона).
6. **Содержимое слотов:** Внутри каждого слота нужно настроить внутренний Flexbox (`align-items: center`, `justify-content: center`) так, чтобы слот был готов принять будущие данные: иконку предмета, цветовую ауру (редкость) и показатель дохода (IPS).
7. Никаких C#-скриптов и хардкода значений в UXML (все размеры и цвета — в USS).

**Критерии приемки (Acceptance Criteria):**
- [x] Обновлен/создан файл `ShelfPanel.uxml`, к нему подключен `Shelf.uss` (и глобальные стили).
- [x] Ограничивающий бокс имеет строгие размеры `586x539`px.
- [x] На фон бокса (или вложенного контейнера) установлено изображение `shelf.png`.
- [x] Настроены эффекты тени/свечения для объема.
- [x] В UXML создано 5 элементов-слотов с общим классом `.shelf-slot`.
- [x] Слоты аккуратно распределены внутри бокса с использованием Flexbox-свойств.
- [x] Каждый слот содержит внутренний Flexbox для центрирования содержимого (иконки, текста, ауры).
- [x] Код не содержит прямого C#-биндинга, только верстку.
